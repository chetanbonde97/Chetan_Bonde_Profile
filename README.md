## Quick Publish (GitHub Pages)
1) Create a new **public** repository on GitHub (e.g., `chetan-portfolio`).
2) Upload all files at the **repo root** (`index.html`, `styles.css`, the PDF, etc.).
3) Go to **Settings → Pages**.
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` • **Folder**: `/ (root)`
4) Save. Wait ~1–2 minutes, then open the URL shown by GitHub.

## Optional
- In `index.html` (JSON-LD), replace `https://<your-username>.github.io/` with your actual URL.
- Add a `CNAME` file if you are using a custom domain.
